## Getting Started

### Development

```bash
# Install dependencies
bun i

# Start client development server
bun run --cwd client dev

# Start server development
bun run --cwd server dev
```

### Deployment

```bash
# Build the project
bun run build

# Deploy to Cloudflare
bun run deploy
```
